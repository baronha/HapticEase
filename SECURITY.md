# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in HapticEase, we kindly ask you to report it privately to the maintainers.

**Please do not disclose the vulnerability publicly until it has been addressed.**

To report a vulnerability, please send an email to [your-email@example.com] with the following details:

- A detailed description of the vulnerability.
- Steps to reproduce the vulnerability.
- Any additional information that might help us understand the issue.

We take all security vulnerabilities seriously and will respond to your report as soon as possible.

## Supported Versions

Currently supported versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.5.x   | :white_check_mark:  |
| < 1.5   | :x:                |

Thank you for helping keep HapticEase secure!
